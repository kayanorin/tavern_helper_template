# 小剧场格式改造方案

## 改造目标

将小剧场从「AI 直接生成完整 HTML」改为「AI 输出带 XML 标签的结构化文本 → 前端脚本正则解析 → 渲染成精美 UI」。

**核心收益：**
- 大幅减少 token 消耗（结构化文本 vs 完整 HTML+CSS+JS）
- 渲染质量可控、风格统一（不再依赖 AI 的 CSS 审美）
- 可维护性强（前端模板集中管理）

---

## 格式设计原则

**参考 `今天吃点啥` 的 `<cooking>` 标签模式：**
- 外层一对 XML 标签包裹整个块（如 `<food_review>...</food_review>`）
- 内部分块用少量 XML 标签标记边界（如 `<restaurant>...</restaurant>`、`<review>...</review>`）
- 块内数据用 `键：值` 冒号格式，一行一个字段
- 需要在一行内塞多个子字段时用 `|` 分隔符

**为什么不用纯 XML：**
- AI 嵌套 XML 标签越多越容易掉格式
- 冒号格式对 AI 来说更自然，不容易出错
- 解析难度也更低（正则逐行 split 即可）

### 2.1 美食点评 `脚本/小剧场渲染/美食点评.js`

**风格**：北欧极简 Scandi（基于 Codex 生成模板改造）
**特性**：标签页切换多家餐厅 · 可交互星级评分 · 食客短评折叠展开 · 喜欢/收藏假交互 · 日/夜主题 · 3档响应式

**Codex 提示词**：
```
Create a restaurant review card component in vanilla HTML/CSS/JS.
Style: Scandinavian minimalist (北欧极简), clean surface with muted earthy tones.

Layout:
- Outer collapsible widget (<details>) with summary header showing title + restaurant count + expand/collapse toggle
- Tab-list for switching between 1-3 restaurants
- Active restaurant panel with:
  - Header: name, cuisine type badge, price pill (¥/¥¥/¥¥¥), score badge
  - Meta row: address pill, hours pill
  - Interactive star rating (5-star, reverse-flex trick for hover cascade)
  - Content grid (2-column on wide, 1-column on narrow):
    Left stack: signature dish card, flavor tag bubbles, action buttons (like/save)
    Right: review list with collapsible review items (<details>)
  - Each review: avatar circle (single char), name, subtitle, static stars, date, toggle, review text

Data-driven: All content from a JS object (appData), renderApp() rebuilds DOM on state change.
Theme: Light/dark mode via CSS custom properties on body[data-theme="night"]. Variables: --canvas, --surface, --surface-strong, --surface-soft, --border, --text, --text-soft, --title, --sage, --sky, --warm, --star.
Responsive: 3 breakpoints (940px max-width, 780px single-column grid, 620px stack all flex containers).
No external dependencies. Chinese UI labels.
```

### T6 — 民间传说（地方传说型）

```
<folklore>
类型：地方传说
地区：炽炎群岛
流传度：广为人知
正文：老一辈说，炽炎群岛最深处的火山口里住着一条蛇，它的舌头分成五段，每一段尝过的东西都不一样——甜的苦的辣的酸的，还有一段什么味道都没有。有个年轻厨师不信邪，爬进去想偷蛇舌头做汤底。三天后他爬出来了，头发全白了，但从此做什么菜都好吃得让人流泪。村里人问他看到了什么，他只说了一句话：「它让我尝了第五段舌头的味道。」后来没人知道第五种味道是什么，但每个去过炽炎群岛的厨师回来后，手艺多少都会变好一点。
注解：炽炎群岛确实是多种高阶火系食材的产地。「第五种味道」在民间常被用来比喻超越四阶、达到法理之口的味觉境界。
</folklore>

### T7 — 民间传说（童谣型）

```
<folklore>
类型：童谣
地区：金麦城
流传度：广为人知
正文：一阶尝个咸和甜，二阶闻出锅里烟。
三阶舌头会说话，四阶一口知前年。
五阶嘛——五阶的事儿，
小孩子别打听，
吃完饭乖乖去洗碗。
注解：这首童谣在金麦城几乎人人会念，用来教小孩认识味阶体系。最后一句「别打听」被认为暗指法理之口（五阶）的能力涉及重大秘密，民间对此讳莫如深。
</folklore>
