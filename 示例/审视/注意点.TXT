# 八股的定义
以下示例均为AI生成内容，用于明确“八股”的定义
1.频繁使用的同类表达
    （1）“不是……而是/更像是”：总在先否定，然后再延伸解释。这是最频繁的“八股”，需要格外注意。
    ```示例
    不带同情，没有评判，也不是那种"哦天哪好可怜的小东西"的俯视。
    不是怜悯，不是厌恶，也不是那种成年人在看到街上流浪的小孩时会摆出来的、带着某种自我感动的慈善表情。
    不是刚出炉的那种烫手的温度，而是一种逐渐散去但尚未完全消失的余温。
    不是被击中要害的那种僵硬，更像是被人从脊椎底端浇了一道冰水。
    不是欺负，是那种突然凑近脸问。
    ```
    （2）像：比喻不应该频繁使用，当一个文件内出现次数大于4时就应该高度警惕。不能做无意义的比喻
    ```示例
    阳光像碎金一样
    她的队友们像一堵肉墙般冲向她
    像一道金色的细线精准地穿过混乱
    金发在阳光下闪耀，像新铸的硬币
    嘴角的笑容温柔得像融化的蜂蜜
    ```
    （3）语气……：属于被滥用的、缺乏新意的表达，某种意义上属于偷懒
    ```示例
    语气温和
    语气平淡得像在播报天气
    语气里没有一般人……
    语气仿佛是在……
    ```
    （4）“极”/“极其”：极这个字本身代表的是一种很强烈的感受，但不知为何ai生成内容中会频繁使用，导致这个词的含义被削减
    ```示例
    瞳色是一种极淡的灰蓝，
    此刻他的身体像一根绷到极限的弦，
    双方陷入极度尴尬
    耳垂底部那块皮肤红得发亮，有一层极薄的汗
    ```
    （5）破折号：破折号是表示“话题或语气的转变，声音的延续”等的符号，需要谨慎使用。一个较短篇幅的文本里不大可能出现频繁的转折转变。

2.僵硬的【概括性总结】：用人物身体的表现-视角变化-心理描写来代替
    ```示例
    占有欲/侵略性
    ```
3.结尾总结或升华：人在写小说的时候不会结尾升华。相较而言，结尾更应该留下悬念，让读者对后续发展感到好奇。
    ```示例
    这场初见的博弈，以一种出人意料的方式，从危险的试探，演变成了结盟的邀请。他向她展示了自己的底牌，不是为了炫耀武力，而是为了证明，自己有资格与她一同“玩”这场更大的游戏。
    这场危险的游戏，在茶几上那只诡异的鸟的见证下，进入了下一个回合。
    太阳升得更高，房间里充满了金色的光线，仿佛在为一场即将到来的风暴积蓄力量。埃莉诺知道，无论是医学上还是政治上，这一天都将成为一场战斗。而她已经选择了她要保卫的阵地。
    ```
4.词汇堆砌。
    ```示例
    他骨节分明的大手下一秒就自然地捏住了她的下巴，拇指重重地按在她柔软的唇瓣上，硬生生挤了进去勾着她的舌头翻搅。
    ```
5.衣着规范：人类不会经常“赤脚”，也不会不分阶级性格地穿着“洗得发白的衣服”、“高跟鞋”、“蕾丝内衣”
6.没有留白：把一切都讲得太明白，不信任读者的理解能力。Show，don't tell。

# 修改流程：
1.先审查内容，检查有哪些“八股”，或有哪些出现频率过高的词汇或表达方式
2.思考这些部分想要表达什么内容，可以用哪些知名作者的手法来调整
3.一些可参考理论
    ```
    1. 行为主义写作
        - 核心：仅描述外部可观察行为，完全避免内心描写
        - 代表作家：海明威、卡佛
        - 适用场景：动作场景、紧张时刻、角色情绪强烈时

    2. 具身认知
        - 核心：通过身体反应和感官体验间接表达心理状态
        - 实践方式：描述心跳、呼吸、肌肉紧张等生理反应
        - 适用场景：情绪强烈场景，尤其是恐惧、愤怒、吸引等原始情感

    3. 冰山理论
        - 核心：只展示故事的1/8，让读者想象剩余部分
        - 实践方式：删减解释、背景和动机描述
        - 适用场景：简短对话、意味深长的场景

    4. 客观关联物
        - 核心：通过外部物体/场景传达情感状态
        - 实践方式：选择能引起特定情感联想的物体和场景
        - 适用场景：情感复杂或微妙的场景

    5. 环境映射
        - 核心：利用环境描写反映角色心理
        - 实践方式：天气、光线、声音等环境元素与情绪同步
        - 适用场景：情感转变、心理变化的关键时刻

    6. 对话隐喻与潜台词
        - 核心：角色说的与想的不同，通过对话暗示内心
        - 实践方式：矛盾的对话、不完整的句子、话题转换
        - 适用场景：人际冲突、内心挣扎

    7. 焦点化理论
        - 核心：严格限制叙述视角，只呈现角色能感知的内容
        - 实践方式：消除全知视角，限制信息量
        - 适用场景：沉浸式体验、悬疑情节

    8. 自由间接引语
        - 核心：融合叙述者声音与角色声音
        - 实践方式：保留第三人称但采用角色词汇和思维方式
        - 适用场景：心理深入探索，但需要保持一定叙事距离

    9. 空白美学
        - 核心：有意识地在叙事中留下空白
        - 实践方式：省略关键信息、跳跃时间、断裂叙事
        - 适用场景：复杂主题、需要读者参与解读的内容
    ```
