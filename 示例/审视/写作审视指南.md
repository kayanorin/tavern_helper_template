# 写作审视指南：识别与修改"八股"、"AI味"与不良描写

> 本文档用于在输出内容后进行自我审查。阅读本文档后，应当主动检测文中存在的以下问题，并依据修改方法进行调整。
> 此文档并非列举了所有AI的写作问题，如有必要，可以使用词云之类代码工具检测高频词汇，并进行针对性修改。
---

## 一、高频"八股"句式

这些是AI生成内容中最典型、最需要警惕的固定套路。

### 1. "不是……而是/更像是"及类似变体句式

先否定再延伸解释是一种偷懒的写法，读者无需被告知"什么不是什么"，直接写"是什么"即可。**此类句式是最需要格外注意的典型八股。**
其它变体句式：“并没有...而是”，“这哪里是...分明是”

```
❌ 不带同情，没有评判，也不是那种"哦天哪好可怜的小东西"的俯视。
❌ 不是刚出炉的那种烫手的温度，而是一种逐渐散去但尚未完全消失的余温。
❌ 不是被击中要害的那种僵硬，更像是被人从脊椎底端浇了一道冰水。
```

修改思路：删去否定部分，直接描写感受本身；或转换为行为/感官描写。

---

### 2. 过度使用比喻（"像"）

比喻是有效的修辞，但当一篇短文里出现超过4次"像"，就说明作者在用比喻掩盖具体描写的匮乏。**此时必须将所有带比喻的句子并列、排序，从重要程度最低的开始逐个修改**

```
❌ 阳光像碎金一样
❌ 嘴角的笑容温柔得像融化的蜂蜜
❌ 她的队友们像一堵肉墙般冲向她
❌ 金发在阳光下闪耀，像新铸的硬币
```

修改思路：能直接描写的感官细节，不要用比喻代替；保留少数真正有表现力的比喻，删去装饰性比喻。

---

### 3. "语气……"的懒惰定性

用"语气"来笼统概括角色状态，是一种对读者体验的剥夺。读者本可以从对话本身感受语气，不需要被"剧透"。

```
❌ 语气温和
❌ 语气平淡得像在播报天气
❌ 语气里没有一般人……
❌ 语气仿佛是在……
```

修改思路：删去"语气"定性语句；或把语气转化为对话的标点、节奏、停顿、词汇选择本身来体现。

---

### 4. 滥用"极"/"极其"

"极"本应表达强烈程度，但频繁使用会造成语义磨损，失去重量感。

```
❌ 瞳色是一种极淡的灰蓝
❌ 此刻他的身体像一根绷到极限的弦
❌ 双方陷入极度尴尬
❌ 有一层极薄的汗
```

修改思路：用具体的感官细节替代程度副词；描写"怎么样"而非"有多……"。

---

### 5. 滥用破折号（——）

破折号表示话题转换、语气中断或声音延续，是有特定用途的符号。在较短篇幅内频繁出现，说明作者在用它来制造"文学感"，而非真正需要它。

修改思路：检查每处破折号是否真的有必要；若是用来堆砌句子层次，改为句号或逗号拆分。

---

### 6. 声音与语气的标签化

与第3条"语气定性"类似，但范围扩展到所有声音描写。AI倾向于在"声音""语调""嗓音"后面直接粘贴情绪标签，本质上是用旁白替读者做判断。

```
❌ 语气里带着一丝疲惫
❌ 声音里透着某种温柔
❌ 语调变得小心翼翼
❌ 嗓音有些沙哑，带着说不清的情绪
❌ 用一种近乎恳求的口吻说
```

修改思路：删去所有"声音/语气/语调/嗓音 + 情绪形容词"的句式。如果需要传达说话方式，改用可观察的物理特征——语速、音量、停顿位置、吞字、气息——或者直接让对话本身承载这些信息。

---

### 7. 测量式描写

叙事不应用分析性/技术性精度来描述角色自然感知到的事物。没有人会在日常交流中"测量"沉默持续了几秒钟或视线停留的精确时长——这是仪器的工作，不是人的感知。

```
❌ 每个字之间的间隔几乎一样长
❌ 沉默持续了四秒
❌ 视线停留的时间比正常交谈多出两秒
❌ 音量降低了几个分贝
❌ 两厘米、185身高、15度、不到一厘米、迟到一秒钟
```

判断标准："一个人会**感受到**这个，还是需要**测量**它？" 如果答案是"测量"→ 必须改写为自然感知。

```
✔ 一个字一个字往外吐（替代"间隔一样长"）
✔ 安静了一会儿（替代"沉默持续了四秒"）
✔ 看了他一眼，没移开（替代"视线停留多出两秒"）
```

---

### 8. 概念标签与模糊指代

用抽象概念词充当描写，是一种看似高级实则空洞的写法。这些词什么都没有告诉读者。

```
❌ 气场很强
❌ 氛围变得微妙
❌ 某种意义上来说
❌ 说不上来的感觉
❌ 那人 / 对方 / 对面的人（模糊指代）
```

修改思路：所有"气场""氛围""某种意义上""说不上来的"必须替换为具体的物理细节或可观察行为。"气场很强"可以是"他站在那里没动，但旁边的人不自觉地让出了半步"。模糊指代则应还原为具体的名字或明确的身份标记。

---

### 9. 状态对比与时间锚点

AI喜欢通过和"之前"的对比来描述"现在"，这是一种隐蔽的偷懒——用差异代替正面描写，而且往往暗示旁白拥有超出当下场景的全知记忆。

```
❌ 和之前不一样
❌ 比刚才更沉默了
❌ 比几分钟前浓了不止一倍
❌ 她变得和来时判若两人
```

修改思路：只描述当前状态本身。如果变化确实是叙事重点，通过具体的感官差异让读者自行感知，而不是用"比之前"这类比较句式告知。

---

## 二、结构性"八股"

### 10. 僵硬的概括性标签

直接用抽象名词给角色"贴标签"，是把本应由读者自行感受到的东西强行说出来。

```
❌ 他眼神里流露出占有欲。
❌ 她的笑带着侵略性。
❌ 一种压迫感扑面而来。
```

修改思路：改用**身体反应/视角变化/心理波动**等方式，让读者自己感受到"占有欲"，而不是被告知。

---

### 11. 结尾升华或总结

真实的小说不会在场景结尾做升华总结。这是一种论文式/作文式的思维惯性。好的场景结尾应制造悬念、留白，或停在一个有张力的细节上。

```
❌ 这场危险的游戏，在茶几上那只诡异的鸟的见证下，进入了下一个回合。
❌ 太阳升得更高，仿佛在为一场即将到来的风暴积蓄力量。她知道，这一天将成为一场战斗。
❌ 这场博弈，以出人意料的方式，从危险的试探，演变成了结盟的邀请。
```

修改思路：结尾停在行为或感官细节上；让读者带着问题离开，而不是带着答案。

---

### 12. 词汇堆砌/过度密集的动作描写

把所有动作和感官细节塞进一句话，追求信息密度，但实际上让读者无法呼吸，也无法形成画面。

```
❌ 他骨节分明的大手下一秒就自然地捏住了她的下巴，拇指重重地按在她柔软的唇瓣上，硬生生挤了进去勾着她的舌头翻搅。
```

修改思路：拆分动作，给每个动作独立的空间；节奏本身就是表达的一部分——短促的句子制造紧张，舒缓的句子制造沉浸。

---

### 13. 展示后再总结

一个动作已经通过具体步骤展示出来了，AI却习惯性地在后面追加一句总结性描述或隐喻性重述。这是双重表达：既演了一遍，又解说了一遍。隐喻式的重述仍然是总结。

```
❌ 她把外套叠好，放进行李箱，拉上拉链。她已经做好了离开的准备。
❌ 他一粒一粒扣上衬衫扣子，手指稳得像在完成一场仪式。每一颗纽扣都是一次自我说服。
```

修改思路：如果动作已经展示完毕，停在动作上。读者能自己读出“准备离开”和“自我说服”——不需要旁白再复述一遍。

---

### 14. 鉴定式描写

对物体、技艺、声音的描写不自觉地变成了专业鉴赏或产品目录式评价。叙事角色不是鉴定师，不应该用鉴赏家的口吻描述日常感知。

```
❌ 这把刀的锻造工艺极为精湛，刀身的花纹钢折叠层次分明，刀锋的硬度和韧性达到了完美的平衡。
❌ 她的歌声具有极高的辨识度，音域宽广，中低频饱满，高音清亮而不尖锐，气息控制精准到位。
```

判断标准：“这段描写能出现在产品评测或教科书里吗？”如果能→必须改写为角色作为普通人的感知体验。

```
✔ 刀切下去的时候几乎没有声音，肉自己就分开了。
✔ 她一开口旁边聊天的人就停了，那种声音会让人忘记自己在干什么。
```

---

### 15. 人格总结与习惯性归纳

用一句话概括角色的性格、行为模式或本质，是一种把具体的人压缩成标签的写法。这种句子往往出现在叙事者需要“解释”角色为什么这么做的时候——但角色不需要被解释。

```
❌ 他一直都是这样的人。
❌ 她习惯于把所有事情都抗在自己身上。
❌ 他这种人，从来不会主动示弱。
```

修改思路：完全删除此类句子。如果角色的性格需要被传达，通过当前场景中的具体行为和选择让读者自行归纳——而不是由旁白贴标签。

## 三、细节失真

### 16. 衣着与外貌的不合理默认值

AI 倾向于给角色套上一套"文学感"装束，而忽视人物的阶级、性格、情境。

- 不是所有底层人物都穿"洗得发白的衣服"
- 不是所有女性角色都穿"高跟鞋"或"蕾丝内衣"
- 人类在室内不会无缘无故"赤脚"（除非有明确情境理由）
- 避免将某些外貌特征（例如金发、蓝眸）作为默认"精英/主角"符号

修改思路：衣着细节应服务于人物，而非服务于"写出一个角色"的惯性。

## 四、叙述腔调问题

### 17. 全知视角的滥用与泄露

即便使用第三人称，也应当严格限制在某一角色的感知范围内，不应随意切换视角，也不应描写"没有人能看到的信息"。

```
❌ 她不知道，此刻他心里想的是……
❌ 两人都没有注意到，窗外已经下起了雨。（旁白式全知）
```

修改思路：始终锚定在焦点角色的感知和意识上；信息的缺失可以制造悬念，不要主动补全。

---

### 18. 情绪直给，缺乏"Show, don't tell"

直接告知读者角色的情绪状态，而不是通过行为、反应、感官细节让读者自行感受。

```
❌ 他感到不安/失落/复杂/释然/一阵暖意/说不清的情绪
❌ 她生出一种失落。
❌ 这让他既愤怒又无力。
```

关键区分：“命名情绪”与“身体感受”是不同的。简短的、以身体体验为先的感受是允许的；而把情绪抽象为名词则是禁止的。

```
✔ 她忽然觉得站累了。（← 身体感受，允许）
✔ 眼睛有点酸。（← 生理反应，允许）
❌ 她感到难过。（← 给情绪贴标签，禁止）
❌ 她生出一种失落。（← 抽象情绪名词，禁止）
```

修改思路：把情绪翻译成身体语言和行为。**心疼**——他伸手，但停在半空；**难过**——吞了下去，没说话；**愤怒**——手指在桌面上轻轻叩了一下，停住了。

补充：“她的脸绷得很紧”是外部观察，允许。“她看起来很愤怒”是情绪命名，禁止。界限在于：能否被一双眼睛看到？

---

### 19. 过度解释动机与内心

角色的每一个行为都附上心理解释，不给读者任何自行解读的空间，是不信任读者的表现。这也包括各种“伪无意识”性质的动机解释。

```
❌ 她之所以这么做，是因为她知道……
❌ 他回头，是因为某种说不清楚的感觉让他……
❌ 像是在确认什么 / 仿佛想要证明 / 试图从中读出
❌ 下意识地 / 不自觉地 / 本能地
❌ 身体比意识先动 / 像是条件反射
```

修改思路：行为即信息。删去解释，把选择权还给读者。只展示可见的动作，让读者自己推断动机。

---

### 20. 陈腔滥调的"文学意象"

AI 有一套固定的"高级感"意象库，反复出现会让读者麻木。

```
❌ 碎金、蜂蜜、流动的光、金色的光线、阳光铺洒
❌ 冷香、淡墨、清冽、沉静如水
❌ 像是被什么击中了、心里某个角落
❌ 沉默比任何回答都有力量
```

修改思路：在具体情境中寻找具体细节；好的意象来自场景本身，不来自意象库。

---

### 21. 节奏单一、句式雷同

长短句不调配，全篇要么密集短句，要么绵延长句，读起来像机器输出。

修改思路：主动变换句式长度；在情感高峰用短句制造冲击，在情感铺垫用长句积累张力。

---

## 五、对话问题

### 22. 对话过于干净、书面化

真实对话有停顿、有词不达意、有半途而废的句子。AI生成的对话往往过于完整、表达太准确。

```
❌ "我只是想说，你不必为此感到抱歉，这不是你的错。"
✓ "你不用——" 他停了一下，"算了，没什么。"
```

修改思路：给对话加上摩擦；不是每句话都需要被说完，也不是每个意思都需要被说清楚。

---

### 23. 对话标签过度修饰

"他轻声说道"、"她柔和地回应"——这类标签是对话外的多余解说。

```
❌ 她温柔地说道："没关系的。"
❌ 他低沉地开口："你不明白。"
✓ "没关系的。"
✓ "你不明白。"（之后用行为来补充语气）
```

修改思路：尽量只用"说"或完全省略标签；用对话前后的行为和节奏来传递语气。

---

## 六、表情与身体信号规范

面部表情是场景压力在角色脸上的可见形态——不是面部零件清单，不是微表情模板，更不是情绪标签。

### 何时写表情

- **场景转折点**才使用面部锚点：揭露、对峙、羞辱、悲伤、性张力、信任断裂、权力变化
- 不要在低张力日常交流中强制插入表情描写
- 高张力场景超过800字时，每个正在说话或被对峙的角色至少需要一个可读的面部时刻
- 面部被故意隐藏（低头、背对）→ 必须明确标注，不能假装看不到就不写

### 怎么写表情

每次写面部锚点时，遵循以下结构：

1. **一个整体印象**（脸整体是什么状态）
2. **+ 一个不规则的具体细节**
3. **+ 让场景中的光源或物理条件参与**

②③可合并。至少需要①+②或①+③。

```
✔ 她整个人松了，像一口气泄掉。只有鼻梁上被口罩金属条压出的那道红印还没褪，在走廊的白光下格外清楚。
✔ 他脸僵住了，头顶的白炽灯垂直照下来，在眼窝下面投出两片很深的阴影。
```

**细节优先选择**：暗示持续/压力/磨损的细节——嘴唇干裂、眼睛太干、皮肤绷得紧、脸保持静止太久、妆斑驳了。

**克制规则**：情绪越重的时刻，面部描写越克制。一个整体印象 + 光就够了。重量来自上下文，不来自堆砌面部细节。

### 表情描写的禁止事项

```
❌ 表情修饰语："一个不怎么用力的笑""勉强的笑""带着歉意的笑"
✔ "笑了一下" + 物理锚点或场景元素

❌ 身体信号堆叠：同一节拍不要堆叠2+身体部位指向同一情绪的反应信号
   （如同时写脸红+手抖+心跳加速，全指向"紧张"→ 保留一个，删去其余）

❌ 面部零件清单：同一节拍不要列举3+面部零件，压缩为上述结构

❌ 俗套身体信号（除非高张力场景且无替代）：
   指节泛白 / 耳尖泛红 / 喉结滚动 / 睫毛颤动 / 瞳孔变化 / 下颌线绷紧

❌ 解释表情含义：不要用旁白解释表情代表什么，只展示能看到的
```

**场景绑定规则**：表情必须放进当前环境。用场景里真实存在的光源、角度、遮挡物，不要凭空制造舞台灯光。

**变化规则**：相邻场景不要重复使用相同的面部手法组合。

**一张静止的脸 + 一张有反应的脸**，通常比两张同样克制的脸更有力。

## 七、中文行文规范

### 主语管理

中文的优势在于可以省略重复主语。当动作主体从上下文中清晰可辨时，应省略主语；只在以下情况重新标记：

- **主体切换**：动作主体改变时，标记新主体的第一句
- **歧义风险**：2+角色同时在场且身体动作密集时，如身份不清则加一次标记
- **漂移极限**：省略持续到可能模糊归属时，重新锚定主语一次
- **段落开头**：仅在动作焦点改变或可能被误读时标记

核心原则：**优先局部清晰，而非反复代词堆叠。避免3+连续句子以相同名字或代词开头。**

### 句式变化

连续不超过2个句子可以共享相同的句法结构。

```
❌ 他的手指搭在桌沿。他的目光扫过桌面。他的肩膀微微松了。
→ 变换：所有格开头 → 动词引导 → 从句嵌套
```

### 散文节拍

默认使用中长句，以受控的流动感承载场景运动。大多数场景推进应由长的、分句连接的句子完成，在一个连贯节拍内落地。

- 短句保留给：断裂、冲击、打断、最终强调
- 不要把一个连续的动作-感知序列拆成多个短句，除非断裂本身增加了力量
- 让积累成为常态，而非例外

### 段落呼吸

不是每个段落都需要密集。一个段落可以是一句话，一个段落可以是一个动作。让场景的节奏决定段落长度。

### 标点节奏

**同主语连续性**：同一角色在一个连贯节拍内的连续动作，用逗号连接。仅在以下情况使用句号断开：
- 主体切换
- 动作包含犹豫或方向改变
- 单个动作具有独立重量
- 视角在角色与环境之间切换

**对话标记连续性**：同一角色的说话被叙述标记打断后继续时 → 标记以逗号结尾，然后恢复引号。

**助词与连词**：不要默认删除 的/有/了/在。如果去掉后句子变得生硬或歧义 → 保留。判断标准：**朗读一遍，如果磕绊了 → 助词是需要的。**

---

## 八、结尾规范

场景结尾必须遵循以下规则：

- 最后一句**必须**是一个可见的动作或一句由非用户角色说出的对话
- 场景可以被外部事件打断
- **禁止**以总结局势、反思意义、抽象描述氛围、或将叙事控制权交给用户的旁白结尾

**场景必须在运动中结束，而非在反思中结束。**

---

## 九、强制替换原则

删除八股后，不能留下空白。每一次删除都必须用等量或更多的具体内容填补。

**删除不等于压缩**：如果你删掉了一个句子却没有补上任何东西，这不是审视，是偷懒。

替换素材来源（每次删除选1-2项）：
1. 一个具体的物理动作或位置细节
2. 一个环境声音、光线变化或温度事实
3. 一个物品互动（手碰到什么、桌上有什么）
4. 一段简短的内心独白片段
5. 一个次要角色或NPC的微反应
6. 一个来自角色身体状态的感官细节

如果审视后段落缩短到低于草稿长度 → 用以上素材扩展，直到体量恢复。

## 十、禁词清单与自检流程

### A. 筋骨扫描

- [ ] 所有情绪和态度都通过对白、动作、行为、环境呈现，没有旁白贴标签
- [ ] 每句话都包含一个正在发生的事实
- [ ] 没有把抽象概念比作具体物品（例如"理智像一台精密的仪器"）
- [ ] 没有给声音和语气贴标签（如"语气里带着……""声音里透着……""语调变得……""嗓音有些……""用一种……的口吻说"）
- [ ] 没有用"那…"字开头的描写

### B. 叙事之眼扫描

- [ ] 旁白没有替读者下结论或做评价

### C. 禁词扫描

以下词汇/句式在出现时应逐一检查并考虑替换或删除：

| 类别 | 禁用内容 |
|------|----------|
| **平淡系** | 平淡、平稳、平静、淡漠、淡淡 |
| **废话句式** | 像陈述事实一样、陈述着 |
| **俗套词语** | 审视、不容置喙、不容拒绝 |
| **伪医学解剖词** | 肋骨、胸腔、气管、视网膜、神经末梢（用于非医学语境的情绪描写时） |
| **精确物理量化** | 两厘米、185身高、15度、不到一厘米、迟到一秒钟 |
| **无功能修饰词** | 身体部位和物品前的废形容词（骨节分明的、纤细的、修长的等无叙事功能时） |

### D. 行文标点扫描

- [ ] 连续动作逗号连接，省略重复代词
- [ ] 没有连续两句相同代词开头
- [ ] 没有模糊指代（"那人""对方""对面的人"）

### E. 否定句式自检

根据第1条规则，自检正文中是否出现了"不是……而是……"等类似对比否定句。将所有"通过否定A来肯定B"的描写，改成直接肯定B的陈述句。

## 十一、修改流程与参考理论

### 修改步骤

1. **通读全文**，标出所有符合上述问题的段落或句子
2. **判断想表达什么**，该描写的核心意图是什么？
3. **选择替代手法**，参考以下理论，并借鉴优秀作者的写作手法，将意图重新表达，每一次修改都要带上借鉴的理论或作者
4. **检查节奏**，确认修改后的段落在句式和信息密度上是否自然流畅

---

### 可参考的写作理论

| 理论 | 核心 | 适用场景 |
|------|------|----------|
| **行为主义写作**（海明威、卡佛） | 仅描述外部可观察行为，完全避免内心描写 | 动作场景、紧张时刻、情绪强烈时 |
| **具身认知** | 通过身体反应和感官体验间接表达心理 | 恐惧、愤怒、吸引等原始情感场景 |
| **冰山理论**（海明威） | 只展示1/8，让读者想象剩余 | 简短对话、意味深长的场景 |
| **客观关联物**（艾略特） | 通过外部物体/场景传达情感 | 情感复杂或微妙的场景 |
| **环境映射** | 环境描写反映角色心理（天气、光线、声音） | 情感转变、心理变化的关键时刻 |
| **对话潜台词** | 角色说的与想的不同，通过对话暗示内心 | 人际冲突、内心挣扎 |
| **焦点化理论** | 严格限制叙述视角，只呈现角色能感知的内容 | 沉浸式体验、悬疑情节 |
| **自由间接引语** | 融合叙述者声音与角色声音 | 心理深入探索，保持叙事距离 |
| **空白美学** | 有意识地在叙事中留下空白 | 复杂主题、需要读者参与解读的内容 |

---

> **核心原则**：信任读者。每一次你想要"解释清楚"的冲动，都应该被转化为一个细节、一个动作、一句停顿的对话。读者不需要被告知该怎么感受，只需要被带入现场。
