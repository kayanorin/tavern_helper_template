# 写作审视指南：识别与修改"八股"、"AI味"与不良描写

> 本文档用于在输出内容后进行自我审查。阅读本文档后，应当主动检测文中存在的以下问题，并依据修改方法进行调整。

---

## 一、高频"八股"句式

这些是AI生成内容中最典型、最需要警惕的固定套路。

### 1. "不是……而是/更像是"句式

先否定再延伸解释是一种偷懒的写法，读者无需被告知"什么不是什么"，直接写"是什么"即可。**此类句式是最需要格外注意的典型八股。**

```
❌ 不带同情，没有评判，也不是那种"哦天哪好可怜的小东西"的俯视。
❌ 不是刚出炉的那种烫手的温度，而是一种逐渐散去但尚未完全消失的余温。
❌ 不是被击中要害的那种僵硬，更像是被人从脊椎底端浇了一道冰水。
```

修改思路：删去否定部分，直接描写感受本身；或转换为行为/感官描写。

---

### 2. 过度使用比喻（"像"）

比喻是有效的修辞，但当一篇短文里出现超过4次"像"，就说明作者在用比喻掩盖具体描写的匮乏。**此时必须将所有带比喻的句子并列、排序，从重要程度最低的开始逐个修改**

```
❌ 阳光像碎金一样
❌ 嘴角的笑容温柔得像融化的蜂蜜
❌ 她的队友们像一堵肉墙般冲向她
❌ 金发在阳光下闪耀，像新铸的硬币
```

修改思路：能直接描写的感官细节，不要用比喻代替；保留少数真正有表现力的比喻，删去装饰性比喻。

---

### 3. "语气……"的懒惰定性

用"语气"来笼统概括角色状态，是一种对读者体验的剥夺。读者本可以从对话本身感受语气，不需要被"剧透"。

```
❌ 语气温和
❌ 语气平淡得像在播报天气
❌ 语气里没有一般人……
❌ 语气仿佛是在……
```

修改思路：删去"语气"定性语句；或把语气转化为对话的标点、节奏、停顿、词汇选择本身来体现。

---

### 4. 滥用"极"/"极其"

"极"本应表达强烈程度，但频繁使用会造成语义磨损，失去重量感。

```
❌ 瞳色是一种极淡的灰蓝
❌ 此刻他的身体像一根绷到极限的弦
❌ 双方陷入极度尴尬
❌ 有一层极薄的汗
```

修改思路：用具体的感官细节替代程度副词；描写"怎么样"而非"有多……"。

---

### 5. 滥用破折号（——）

破折号表示话题转换、语气中断或声音延续，是有特定用途的符号。在较短篇幅内频繁出现，说明作者在用它来制造"文学感"，而非真正需要它。

修改思路：检查每处破折号是否真的有必要；若是用来堆砌句子层次，改为句号或逗号拆分。

---

## 二、结构性"八股"

### 6. 僵硬的概括性标签

直接用抽象名词给角色"贴标签"，是把本应由读者自行感受到的东西强行说出来。

```
❌ 他眼神里流露出占有欲。
❌ 她的笑带着侵略性。
❌ 一种压迫感扑面而来。
```

修改思路：改用**身体反应/视角变化/心理波动**等方式，让读者自己感受到"占有欲"，而不是被告知。

---

### 7. 结尾升华或总结

真实的小说不会在场景结尾做升华总结。这是一种论文式/作文式的思维惯性。好的场景结尾应制造悬念、留白，或停在一个有张力的细节上。

```
❌ 这场危险的游戏，在茶几上那只诡异的鸟的见证下，进入了下一个回合。
❌ 太阳升得更高，仿佛在为一场即将到来的风暴积蓄力量。她知道，这一天将成为一场战斗。
❌ 这场博弈，以出人意料的方式，从危险的试探，演变成了结盟的邀请。
```

修改思路：结尾停在行为或感官细节上；让读者带着问题离开，而不是带着答案。

---

### 8. 词汇堆砌/过度密集的动作描写

把所有动作和感官细节塞进一句话，追求信息密度，但实际上让读者无法呼吸，也无法形成画面。

```
❌ 他骨节分明的大手下一秒就自然地捏住了她的下巴，拇指重重地按在她柔软的唇瓣上，硬生生挤了进去勾着她的舌头翻搅。
```

修改思路：拆分动作，给每个动作独立的空间；节奏本身就是表达的一部分——短促的句子制造紧张，舒缓的句子制造沉浸。

---

## 三、细节失真

### 9. 衣着与外貌的不合理默认值

AI 倾向于给角色套上一套"文学感"装束，而忽视人物的阶级、性格、情境。

- 不是所有底层人物都穿"洗得发白的衣服"
- 不是所有女性角色都穿"高跟鞋"或"蕾丝内衣"
- 人类在室内不会无缘无故"赤脚"（除非有明确情境理由）
- 避免将某些外貌特征（例如金发、蓝眸）作为默认"精英/主角"符号

修改思路：衣着细节应服务于人物，而非服务于"写出一个角色"的惯性。


---

## 四、叙述腔调问题

### 10. 全知视角的滥用与泄露

即便使用第三人称，也应当严格限制在某一角色的感知范围内，不应随意切换视角，也不应描写"没有人能看到的信息"。

```
❌ 她不知道，此刻他心里想的是……
❌ 两人都没有注意到，窗外已经下起了雨。（旁白式全知）
```

修改思路：始终锚定在焦点角色的感知和意识上；信息的缺失可以制造悬念，不要主动补全。

---

### 11. 情绪直给，缺乏"Show, don't tell"

直接告知读者角色的情绪状态，而不是通过行为、反应、感官细节让读者自行感受。

```
❌ 他感到一阵心疼。
❌ 她突然有些难过。
❌ 这让他既愤怒又无力。
```

修改思路：把情绪翻译成身体语言和行为。**心疼**——他伸手，但停在半空；**难过**——吞了下去，没说话；**愤怒**——手指在桌面上轻轻叩了一下，停住了。

---

### 12. 过度解释动机与内心

角色的每一个行为都附上心理解释，不给读者任何自行解读的空间，是不信任读者的表现。

```
❌ 她之所以这么做，是因为她知道……
❌ 他回头，是因为某种说不清楚的感觉让他……
```

修改思路：行为即信息。删去解释，把选择权还给读者。

---

### 13. 陈腔滥调的"文学意象"

AI 有一套固定的"高级感"意象库，反复出现会让读者麻木。

```
❌ 碎金、蜂蜜、流动的光、金色的光线、阳光铺洒
❌ 冷香、淡墨、清冽、沉静如水
❌ 像是被什么击中了、心里某个角落
❌ 沉默比任何回答都有力量
```

修改思路：在具体情境中寻找具体细节；好的意象来自场景本身，不来自意象库。

---

### 14. 节奏单一、句式雷同

长短句不调配，全篇要么密集短句，要么绵延长句，读起来像机器输出。

修改思路：主动变换句式长度；在情感高峰用短句制造冲击，在情感铺垫用长句积累张力。

---

## 五、对话问题

### 15. 对话过于干净、书面化

真实对话有停顿、有词不达意、有半途而废的句子。AI生成的对话往往过于完整、表达太准确。

```
❌ "我只是想说，你不必为此感到抱歉，这不是你的错。"
✓ "你不用——" 他停了一下，"算了，没什么。"
```

修改思路：给对话加上摩擦；不是每句话都需要被说完，也不是每个意思都需要被说清楚。

---

### 16. 对话标签过度修饰

"他轻声说道"、"她柔和地回应"——这类标签是对话外的多余解说。

```
❌ 她温柔地说道："没关系的。"
❌ 他低沉地开口："你不明白。"
✓ "没关系的。"
✓ "你不明白。"（之后用行为来补充语气）
```

修改思路：尽量只用"说"或完全省略标签；用对话前后的行为和节奏来传递语气。

---

## 六、修改流程与参考理论

### 修改步骤

1. **通读全文**，标出所有符合上述问题的段落或句子
2. **判断想表达什么**，该描写的核心意图是什么？
3. **选择替代手法**，参考以下理论，并借鉴优秀作者的写作手法，将意图重新表达，每一次修改都要带上借鉴的理论或作者
4. **检查节奏**，确认修改后的段落在句式和信息密度上是否自然流畅

---

### 可参考的写作理论

| 理论 | 核心 | 适用场景 |
|------|------|----------|
| **行为主义写作**（海明威、卡佛） | 仅描述外部可观察行为，完全避免内心描写 | 动作场景、紧张时刻、情绪强烈时 |
| **具身认知** | 通过身体反应和感官体验间接表达心理 | 恐惧、愤怒、吸引等原始情感场景 |
| **冰山理论**（海明威） | 只展示1/8，让读者想象剩余 | 简短对话、意味深长的场景 |
| **客观关联物**（艾略特） | 通过外部物体/场景传达情感 | 情感复杂或微妙的场景 |
| **环境映射** | 环境描写反映角色心理（天气、光线、声音） | 情感转变、心理变化的关键时刻 |
| **对话潜台词** | 角色说的与想的不同，通过对话暗示内心 | 人际冲突、内心挣扎 |
| **焦点化理论** | 严格限制叙述视角，只呈现角色能感知的内容 | 沉浸式体验、悬疑情节 |
| **自由间接引语** | 融合叙述者声音与角色声音 | 心理深入探索，保持叙事距离 |
| **空白美学** | 有意识地在叙事中留下空白 | 复杂主题、需要读者参与解读的内容 |

---

> **核心原则**：信任读者。每一次你想要"解释清楚"的冲动，都应该被转化为一个细节、一个动作、一句停顿的对话。读者不需要被告知该怎么感受——他们只需要被带入现场。
